# Seminario Evangelios Sinópticos 3

Aplicación web interactiva para la presentación del seminario "Evangelios Sinópticos" del LADP (Legión de Apóstoles de Dios y Postreros).

Construida con React + TypeScript + Vite. Desplegada en Vercel.

## Stack

- **React 19** + **TypeScript**
- **Vite 8** (build tool)
- **Tailwind CSS 4** (estilos)
- **Framer Motion** (animaciones)
- **Zod** (validación de tipos)

## Scripts

```bash
npm run dev      # Servidor de desarrollo
npm run build    # Build de producción
npm run preview  # Preview del build
```

## PDF

Para regenerar el PDF del seminario:

```bash
npm run generate-pdf
```

Requiere Chrome instalado en `C:\Program Files\Google\Chrome\Application\chrome.exe`.
